# 🚗 Road Safety & Crash Analysis Workflow

This project analyzes urban traffic crash data from three major cities—Austin, Chicago, and New York—to understand accident patterns, data quality, and key safety insights.

---

## 📌 Overview

* Analyzed **3.2 million+ records** across **131 variables**
* Focused on crash patterns, injuries, fatalities, and contributing factors
* Combined data profiling, transformation, and visualization techniques

---

## 🎯 Objectives

* Analyze crash data across multiple cities
* Ensure data quality through transformation and profiling
* Generate insights using dashboards and visual reports

---

## 🛠️ Tools & Technologies

* **Alteryx** – Data preparation and initial analysis
* **Talend** – Data transformation and cleaning
* **Power BI** – Interactive dashboards
* **Tableau** – Data visualization and reporting
* **Python (ydata-profiling)** – Dataset profiling

---

## 📂 Project Components

* Data profiling notebooks (Austin, Chicago, NYC)
* Profiling reports (PDF summaries)
* Data mapping document
* Dimensional data model

---

## 📊 Key Analysis Performed

* Accident counts across cities
* Identification of high-risk areas
* Injury and fatality analysis
* Pedestrian involvement analysis
* Time-based analysis (seasonality, weekdays vs weekends)
* Contributing factors in accidents
* Spatial and regional accident distribution

---

## 🔍 Key Insights

* Majority of accidents are non-fatal but frequent
* Missing and inconsistent data require preprocessing
* Certain areas and time periods show higher accident rates
* Pedestrian-related incidents highlight critical safety concerns

---

## 🎯 Outcome

This project demonstrates how data profiling, transformation, and visualization can be combined to generate meaningful insights that support:

* Road safety improvements
* Data-driven policy decisions
* Urban traffic planning
